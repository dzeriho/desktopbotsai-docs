# Generate AI images

To generate a image, choose a bot that only generates images, or talk with a bot and tell him to make to generate images for you from the replies.

1. Select `Generate Images` bot
2. This bot is a raw bot to generate images
3. Enter prompt on which the bot will generate an image
4. You can be expressive to explain which style and colors, besides the subject, you expect
